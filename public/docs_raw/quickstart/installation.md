## How a single letter reaches the screen.

Beneath every Letter on this screen lies a hidden world of vector outlines and clever systems.
What looks like a single character is the result of countless steps invisible to the eye.
First, the system reads glyph's geometry — the curves and points that define its true shape
Then it runs tiny `instructions` to nudge each outline so that it lands cleanly on the gird.
Finally. every pixel is shaded with stripes of red, and blue to render edges
All of this happens in milliseconds, every time you scroll every time you blink at the page.
